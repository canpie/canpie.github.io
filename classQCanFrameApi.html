QCanFrameApi Class Reference

Detailed Description

The QCanFrameApi class is used to transfer information between a CAN network (QCanNetwork, i.e. the server) and a CAN socket (QCanSocket, i.e. the client). The information exchange includes the current bit-rate settings, the CAN mode (refer to CAN_Mode_e) and the name of the CAN interface (QCanPlugin).
