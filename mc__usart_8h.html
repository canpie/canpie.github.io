int32_t McUsartGetRcvStatus ( uint8_t  ubPortV)

Get receive status.

Parameters
ubPortV- Serial USART port number
Returns
Error code or number of bytes in receive buffer
See also
McUsartGetTrmStatus()

This function returns the receive status of the serial USART port with the index ubPortV. The number is taken from the enumeration #USART_PORT_e.

A return value of 0 (i.e. eUSART_ERROR_OK) denotes that there is no reception in progress and no data is inside the receive buffer. A negative return value denotes an error. The error code is taken from the enumeration USART_ERROR_e.

A value greater than 0 specifies the number of bytes which are inside the receive buffer.

int32_t McUsartGetTrmStatus ( uint8_t  ubPortV)

Get transmit status.

Parameters
ubPortV- Serial USART port number
Returns
Error code or number of bytes in transmit buffer
See also
McUsartGetRcvStatus()

This function returns the transmit status of the serial USART port with the index ubPortV. The number is taken from the enumeration #USART_PORT_e.

A return value of 0 (i.e. eUSART_ERROR_OK) denotes that there is no transmission in progress and a new transmission can be started by calling McUsartWrite(). A negative return value denotes an error. The error code is taken from the enumeration USART_ERROR_e. A value greater than 0 specifies the number of bytes which are still in the transmit buffer. In this case you can not call McUsartWrite(), as it will return with an error code of eUSART_ERROR_BUSY.

Initialise Serial Interface.

Parameters
ubPortV- Serial USART port number
ulBaudV- Baud rate value
ubModeV- Mode of USART port
Returns
eUSART_ERROR_OK or an error value from USART_ERROR_e
See also
McUsartRelease()

This function initialises the serial USART port with the index ubPortV. The number is taken from the enumeration #USART_PORT_e. The initial baudrate is set via ulBaudV. The number of data bits, stop bits and the parity is configured via the parameter ubModeV.

Attention
After initialisation USART reception and sending are disabled. Call McUsartSetDir() to setup direction of transmission.

Write data on Serial Interface.

Parameters
ubPortV- Serial USART port number
pubBufferV- Pointer to data buffer
uwSizeV- Number of bytes to write

This function writes data to the serial USART port with index ubPortV. The start address of the data to write is given by the parameter pubBufferV, whereas ulSizeV denotes the number of bytes to write.

Attention
This function is not blocking! Setup a transmit callback handler to trigger transmission finish.
