QCanSocketDialog Class Reference

Detailed Description

The QCanSocketDialog class provides a dialog for connection to an existing QCanNetwork. The dialog creates a new QCanSocket and tries to connect after show() is called. It is possible to setup the desired CAN channel by calling setChannel() is advance. Also the CANpie server host address can be configured by calling setHostAddress().

The status of the connection is displayed after the CAN details label.
