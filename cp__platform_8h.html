Detailed Description

The CANpieFD driver can be configured during compile time via several configuration options. They are typically defined in the cp_platform.h configuration file. See CANpie configuration options for a detailed description of all options. If symbols are not defined, they get a default value which is assigned in the canpie.h header file.
