CANpie FD Server Plug-Ins

The CANpie FD Server plug-ins allow to add any kind of physical CAN interface. For an example implementation please refer to the qcan_template directory.

The following table gives an overview of plug-ins available inside the source/qcan/applications/plugins directory.

qcan_ixxat

Plug-in for IXXAT CAN interfaces

qcan_peak

Plug-in for PEAK CAN interfaces

qcan_template

Plug-in example, Template for CAN interface implementation

The tab Information of the CANpie FD Server shows which plug-in is attached to a CAN channel together with version details.
