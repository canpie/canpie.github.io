◆ CpFifoInit

#define CpFifoInit (   FIFO_PTR,
  MSG_PTR,
  SIZE 
)
Value:
do { \
(FIFO_PTR)->ulIndexIn = 0; \
(FIFO_PTR)->ulIndexOut = 0; \
(FIFO_PTR)->ulIndexMax = (SIZE); \
(FIFO_PTR)->ptsCanMsg = (MSG_PTR); \
} while (0)

Initialise FIFO.

Value ctrl have to be a (static) variable from type of a McFifoCtrl_ts. Value size gives number of entries the FIFO contains.

Here is an example for initialisation of FIFO:

...
// define example FIFO
#define NUMBER_OF_FIFO_ENTRIES 20
static McFifoCtrl_ts tsMcFifoCtrlS;
static uint32_t aulUserFifoDataS[NUMBER_OF_FIFO_ENTRIES]; // can be an array of struct
...
// initialise example FIFO
MC_FIFO_INIT(tsMcFifoCtrlS,NUMBER_OF_FIFO_ENTRIES);
Attention
It is important that NUMBER_OF_FIFO_ENTRIES value that was used for atsUserFifoDataS declaration is the same as value that is used with MC_FIFO_INIT();
