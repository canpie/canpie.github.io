QCanServer Class Reference

Detailed Description

This class represents a CAN server, which incorporates up to QCAN_NETWORK_MAX number of CAN networks (QCanNetwork). It is only possible to run one instance of a QCanServer on a machine. In order to avoid multiple instances, the QCanServer class initialises a shared memory region which can be accessed using the QCanServerSettings class.

Inherits QObject.
